# Donate Auth 💙

A simple and modern **Donation Authentication Web App** built with **React**, providing a clean UI for users to authenticate, manage their profiles, and explore donation-related features.  

This project focuses on authentication, responsive design, and a smooth user experience using **Tailwind CSS** and **DaisyUI**.

---

## 🌟 Features

- 🔐 **User Authentication** (Firebase Auth: Email/Password, Google Sign-in)  
- 🎨 **Modern UI** with Tailwind CSS & DaisyUI  
- 📱 **Responsive Design** (works on desktop, mobile & Tab)  
- 🖼️ **Reusable Components** for better scalability  
- 🚀 **Fast & Optimized** with React  
- 🎭 **React Icons Integration** for a polished look  
- ☁️ **Firebase Hosting** (deployed at: [donate-auth.web.app](https://donate-auth.web.app/))  

---

## 🛠️ Tech Stack

- ⚛️ **React.js** – Frontend framework  
- 🎨 **Tailwind CSS** – Styling  
- 🎨 **DaisyUI** – Tailwind CSS component library  
- 🎭 **React Icons** – Icons library  
- 🔥 **Firebase** – Authentication & hosting
